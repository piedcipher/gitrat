# Gitrat :rat:
Gitrat (**Git**hub-T**ra**i**t**ors) is a CLI utility to track GitHub (un)followers.

## Prerequisite
- [Dart](https://www.dartlang.org/)

## Development
```
$ git clone https://github.com/piedcipher/gitrat
$ cd gitrat && pub get
$ dart main.dart <username>
```
